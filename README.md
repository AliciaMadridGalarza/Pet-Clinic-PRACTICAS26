# Pet-Clinic-PRACTICAS26

## 🐾 Pet Clinic — Base de Datos & Conexión JDBC

📘 Descripción del repositorio
Este repositorio contiene los recursos esenciales para iniciar un proyecto de gestión de una clínica veterinaria (Pet Clinic).
Incluye:

El script SQL con la creación de la base de datos y sus tablas.

Un archivo con inserciones de datos para poblar la base.

Una clase Java JDBC para establecer la conexión con MySQL.
